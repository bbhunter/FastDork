FastDork site selector

New domain? Add it to host_permissions in manifest.json around line 44, then reload the extension. If checked, this config appears in the 'Search Engine' dropdown and uses the Base URL for searches & import matching. If unchecked, use URL patterns below for import matching.
(Required if 'Use as Search Engine' is checked. E.g., https://www.google.com/search?q=) (Optional parameters appended to the Base URL. E.g., &filter=0 or &type=Code)
(Required. CSS Selector(s) to find the links/data to extract, comma-separated.) (Optional. CSS Selector(s) for the 'Next' button for pagination, comma-separated.) (Specify whether to extract the link URL ('href') or the visible text ('innerText').)
(Used for 'Import' button if not a Search Engine. One simple text pattern per line. E.g., `hackerone.com/reports/`, `exploit-db.com/search`)
(Optional. One URL substring per line. If the page URL contains one of these patterns, FastDork pauses import as CAPTCHA-blocked.) (Optional. One text snippet per line. Matching is case-insensitive and accent-insensitive.) (Optional. One CSS selector per line for CAPTCHA/challenge elements.)
